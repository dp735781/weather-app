A modern and responsive weather application that provides real-time weather information for any city worldwide. It uses a weather API to fetch live data and displays key details like temperature, humidity, wind speed, and weather conditions in a simple and user-friendly interface.

✨ Features

🔍 Search weather by city name

🌡️ Real-time temperature updates

💨 Wind speed and humidity info

☁️ Dynamic weather conditions display

📱 Fully responsive (Mobile + Desktop)

⚡ Fast and lightweight

CSS

JavaScript

Weather API (e.g., OpenWeatherMap)
